# 订单送达与确认收货设计

## 流转设计

订单从支付成功后的 `处理中` 开始，农场管理员保存物流时自动进入 `已发货`。当物流实际送达后，由管理端手动切换到 `已送达`。消费者只有在订单为 `已送达` 时才可点击“确认收货”，确认后状态变为 `已收货`。

## 权限设计

- 农场管理员/平台管理员：可录入物流、可将订单标记为 `已送达`，但不能直接标记为 `已收货`。
- 消费者：只能对自己的 `已送达` 订单执行“确认收货”。

## 页面设计

- 管理端订单页保留状态下拉，但对 `已收货` 仅做展示，不允许管理员手动切回。
- 消费者订单页新增“确认收货”按钮和成功/失败提示。
- 订单时间线增加“确认收货时间”。
