# 评论权限与商品发布人执行计划

## 执行级别

M 级，单仓库联动改造。

## 实施项

1. 扩展商品模型，记录发布人。
2. 商品查询结果返回发布人名字。
3. 评论资格抽象为统一判定逻辑。
4. 订单接口返回每个商品的评论状态。
5. 前端订单页增加商品级评论入口。
6. 管理后台为发布人显示商品评论。

## 验证命令

- `backend/run-mvn-jdk17.cmd -q -DskipTests compile`
- `frontend/npm run build`

## 回滚规则

- 若商品发布人字段导致查询映射异常，优先回滚发布人关联字段和相关接口扩展。
- 若订单页评论表单影响构建，保留后端资格校验，单独回滚前端订单页交互。
